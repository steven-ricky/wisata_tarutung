Wisata Tarutung – Landing Page React

Landing page modern untuk mempromosikan kota/desa. Murni frontend, siap pakai tanpa backend!

Fitur Utama

React + React Router 7.7 → Navigasi mulus antar halaman

Framer Motion → Animasi keren & interaktif

Headless UI + Lucide Icons → UI modern dan stylish

Google Fonts → Typography kece & readable

Fully responsive → Tampilan rapi di desktop & mobile

Demo

Cara Instalasi & Menjalankan Project

⚠️ Catatan: Project ini tidak ada backend, semua konten statis. Node.js hanya diperlukan untuk development dan build React.

Unzip File
Ekstrak file zip ke folder di komputer kamu.

Install Dependencies
Pastikan sudah punya Node.js v16+ dan npm.
Buka terminal/cmd di folder project, lalu jalankan:

npm install


Jalankan Development Server
Untuk melihat project berjalan di lokal:

npm run dev


Buka browser dan akses:

http://localhost:5173


Build untuk Produksi
Jika sudah siap deploy:

npm run build


Hasil build akan berada di folder:

dist/


Folder dist siap diupload ke hosting seperti Vercel, Netlify, atau hosting lain.

Customisasi Konten

Edit konten di folder src

Ganti gambar di public/assets

Update font & icon sesuai kebutuhan

Tips & Info

Node.js hanya untuk dev & build, bukan backend.

Semua assets sudah termasuk di zip → bisa langsung dipakai.

Cocok untuk website desa/kota, tourism project, atau portfolio.
